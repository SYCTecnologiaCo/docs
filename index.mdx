Folyo es la API de facturación electrónica más directa de Chile. Con una sola petición HTTP emites cualquiera de los 8 tipos de DTE soportados y el documento llega al SII en menos de 2 segundos, sin necesidad de integrar proveedores externos ni gestionar certificados manualmente.

## Cómo funciona

<Steps>
  <Step title="Crea tu cuenta y obtén tu API key">
    Regístrate en [folyo.cl](https://folyo.cl) y accede a tu dashboard para generar tu API key de acceso.
  </Step>
  <Step title="Configura la autenticación">
    Incluye tu API key en el header `Authorization: Bearer <tu_api_key>` en cada petición.
  </Step>
  <Step title="Emite tu primer DTE">
    Envía un `POST` a `api.folyo.cl/v1/dte/emitir` con los datos del documento. Recibirás un `track_id` de confirmación.
  </Step>
  <Step title="Consulta el estado">
    Usa el `track_id` para verificar que el SII recibió y aceptó tu documento.
  </Step>
</Steps>
